# cstrike-bhop-in-linux
Counter-Strike 1.6 Bunnyhop Hack (internal, Linux)

# How to compile?

Run the following commands on the project directory:
```
$ chmod +x compile.sh
$ ./compile.sh
```

# How to inject?
Run the following commands on the project directory:
```
$ chmod +x inject.sh
$ sudo ./inject.sh
```
